# Prompt Engineering Workshop for QA Engineers

Two self-contained HTML files. Open locally or host anywhere.

## Files

- **`index.html`** — Full workshop (intro, ISTQB CT-GenAI 6-component anatomy,
  techniques, use cases, builder, lab, real-model walkthrough, exercises, take-home).
- **`prompt-lab.html`** — Standalone Prompt Lab only. Best for short practice
  sessions or sharing as a focused exercise.

## How to use

- **Local:** double-click either HTML file. Works offline.
- **Internal sharing:** attach to Slack / Quip / Amazon Circles / Internal Wiki.
- **Public sharing:** drop on Netlify / GitHub Pages / S3.

Both files are 100% self-contained — no external scripts, stylesheets, or images.
The only outbound connections are clickable links to free tools (AI Studio,
LMSYS Arena, Hugging Face Chat) which open in a new tab when the user clicks.
